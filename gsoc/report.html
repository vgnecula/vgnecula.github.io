Introduction

This report is the final deliverable for GSoC 2024, showcasing my contributions within GeomScale.

Current algorithms for volume computation of high-dimensional convex bodies, like those used in the VolEsti package, utilize uniform sampling techniques. This project aims to develop a more efficient algorithm by utilizing advancements in non-uniform sampling techniques.

The proposed Hamiltonian Monte Carlo (HMC) methods, as highlighted by Pakman and Paninski, are especially suitable for handling truncated multivariate distributions, a common challenge in high-dimensional contexts. Integration with the VolEsti package will use C++, aligning with high-performance mathematical software development best practices as discussed by Fisikopoulos et al.
